# 📊 PNCP-TO-EXCEL

Uma extensão moderna para Google Chrome desenvolvida para otimizar a extração de dados do **Portal Nacional de Contratações Públicas (PNCP)**.

## ✨ Funcionalidades
* **Extração Inteligente:** Captura dados diretamente de tabelas dinâmicas em Angular.
* **Injeção de Interface:** Botão flutuante minimalista que não interfere na navegação.
* **Geração Offline:** Processamento 100% local com biblioteca `SheetJS`.
* **Privacidade:** Nenhum dado é enviado para servidores externos.

---

## 🛠️ Tecnologias Utilizadas
* **JavaScript (Vanilla)**
* **CSS3 & HTML5**
* **Chrome Extension API (Manifest V3)**
* **SheetJS (xlsx)**

---

## 🚀 Como Instalar
1. Vá até a aba **Releases** deste repositório.
2. Baixe o arquivo **`.crx`** mais recente.
3. No Chrome (`chrome://extensions/`), ative o **Modo do desenvolvedor**.
4. Arraste e solte o arquivo `.crx` na tela.
